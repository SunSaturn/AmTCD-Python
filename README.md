# AmTCD-Python
## ЛР 51. AmTCD (TextCodeDecode) Базовое программирование. Создание программ с GUI
### Задача 1. «Блокнот AmTCD»
Задание: Написать приложение с графическим интерфейсом (application with a graphical interface) «Блокнот AmTCD» (файл приложения: AmTCD). Он должен позволять: создавать/открывать/сохранять зашифрованный тестовый файл, должно быть предусмотрено ввод и сохранение личного ключа, вывод не модальной формы «Справка», вывод модальной формы «О программе».

Предусмотреть выполнение команд файлового меню средствами операционной системы (т.н. «Стандартные диалоги открытия/сохранения файлов»)

Программа должна предусматривать хранение конфигурационной информации в файле «AmTCD.ini» в виде:

|[main]|
|-----------------------|
|keyuser = 2445948689111|


Файл с зашифрованной информацией должен иметь вид «ИмяФайла.txtx», где ИмяФайла — любое пользовательское имя, а формат:

| [main] |
|-------------------------------------------|
|keyopen = 19609668179902086861293819 |
|mess = Ч ойонэ$црвпзн ндпоббмяг8#Зесвжо#имн|
